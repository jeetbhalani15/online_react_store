## Steps to Clone and Run the App

### 1. Clone the Repository

```bash
git clone https://github.com/jeetbhalani15/task-management-app.git
cd repository
```

### 2. Install Project Dependencies
```bash
npm install
```

### 3. Running the Application
Start the development server:
```bash
npm run dev
```
